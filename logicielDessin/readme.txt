
Projet réalisé par Sara Salé, Romain Andres, Marta Boshkovska en L2 Informatique. 


Logiciel de dessin de formes géométriques en C. La réalisation de notre projet ont été fondées sur l'organisation pré-établie et des définitions de méthodes fournies par Julien David, professeur à l'université de Caen Normandie.

___________________________________________________________________________

REQUIRED: Makefile, library SDL for c 
Pour compiler, il faut utiliser Makefile 
Taper: make
Puis pour lancer il faut executer le fichier executale projet 
Pour lancer les tests, il faut faure make test  
Pour lancer séparément un test, faire make nom_test

linux:
make
./projet
